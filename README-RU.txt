Программа учета транпортных средств.
=====================================


Установка:
==========
Для установки запустите exe файл. Установка происходит быстро: на экране посередине появится рисунок, который быстро пропадет. Установка закончена. На рабочем столе появится ярлык.
После первого запуска программы автоматически будет созданы файлы:
1) База Данных:  C:\Users\User\AppData\Local\tos\app-1.0.0\resources\app\database.sqlite
2) Настройки программы:  C:\Users\User\AppData\Local\tos\app-1.0.0\resources\app\properties.json

Логи программа пишет в stdout. При запуске с ярлыка их нет. Логи можно увидеть, если запустить программу из терминала.


Подготовка к сборке из исходников:
=====================
В папке с проектом с исходными кодами выполнить следующие команды в терминале:
> npm install
> npx electron-forge import

Запуск из исходных кодов:
=========================
В терминале:
> npm start

Компиляция и создание готовой программы:
=======================================
На компьютере с ОС Виндовс выполнить команду:
> npm run make

Схему базы данных и первичные данные хранятся в файла: /src/db/*.sql
После завершения работы компилятора появится папка out/make, в которой будет Папка с установщиком.
